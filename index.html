Configuração do Certificado

O arquivo CSV deve conter pelo menos uma coluna com nomes. Colunas como "Nome", "Participante" ou "Aluno" são automaticamente detectadas.
Use [NOME] onde o nome do participante será inserido automaticamente.
Recomendado: imagem PNG ou JPG com dimensões A4 (297×210mm) em alta resolução (300 DPI).

Ajuda - Como Usar
Formato do CSV

O arquivo CSV deve conter uma coluna com os nomes dos participantes. Exemplo:

Nome
João Silva
Maria Santos
Pedro Costa
Texto Personalizado

Use [NOME] no texto onde o nome do participante deve aparecer.

Imagem de Fundo

Use uma imagem com resolução mínima de 2480×3508 pixels (A4 em 300 DPI) para melhor qualidade.

A imagem de fundo já deve conter o layout completo e assinatura do resposável.

Dica: Todos os dados são processados localmente no seu navegador. Nenhuma informação é enviada para servidores externos.
